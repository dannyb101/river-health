Spill Data

CSO spill data is added to a calculation by uploading a .csv file. These .csv files must be formatted in one of two ways. Note that all columns with time in them must be formatted in the same way as pictured and that the units of flow must be m3/s too.

The first format has one column for time and one or more columns for spill flow. This is shown in the image below. For this format, there can be any number of header rows and they can be called what you like.


The second acceptable format of spill data is pictured below. It has two columns for time and one or more columns for spill flow. The format of the time in the first column is the same as in the csv above. The format of the time in the second column is in seconds. As in the image, the second column title must be ‘Seconds’. Below that row with ‘Seconds’ in, there may be any number of header rows and they can be called what you like.
